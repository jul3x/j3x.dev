--- date: 2020-01-23 img_pos: center name: Latte Compiler img: latte.webp short: | Compiler for the object-oriented Latte language to x86 ASM project_url: https://github.com/jul3x/LatteCompiler --- A simple compiler for the Latte programming language (a statically typed language with object-oriented features) to x86 GAS assembly. The language is syntactically and semantically similar to Java. A description of its context-free grammar can be found in the GitHub repository.

Along with the language syntax, the compiler is a fully functional development tool that allows the writing of complex programs. The repository contains dozens of test programs and automatic functional tests (including output comparison).

Implemented language features:
* arithmetic operations
* string operations
* conditional operations and Boolean logic
* lazy evaluation
* loops
* arrays
* null objects
* type casting
* structures
* classes and methods
* multiple inheritance

The compiler supports:
* syntactic analysis
* semantic analysis
* type checking
* garbage collection
* control flow and infinite loop detection
* constant folding for simple arithmetic expressions
* constant folding for simple logical expressions
* removal of redundant code (push/pop and pop/push sequences)
* removal of unreachable code
The project was developed during my master's studies.

As an introduction to the world of compilers, I also implemented a simple compiler for arithmetic expressions to JVM bytecode and LLVM bitcode. You can see the code here.
