While collaborating with the Talent Association, I designed preparatory problem sets for the Junior Olympiad in Informatics (OIG).

The problems consisted of creatively masked algorithmic problems using colorful stories. The task solver had to analyze the story and formulate a mathematical problem. Using the example queries and answers provided in the problem, the solver was then required to propose a computer program that, while meeting the specified conditions (such as maximum number values and memory-time constraints), would solve the problem for a set of test cases. A particular challenge in the proposed problems was to find a solution with appropriately low computational complexity.

I prepared four sets of problems along with sample solutions that address these problems. The problem statements (in Polish) and the code (written in C++) can be accessed through the links below:

  • Napoleon's Army: problem statement, solution.
  • Biological Bomb: problem statement, solution.
  • Energy Loss: problem statement, solution.
